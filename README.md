## Install it from PyPI

```bash
pip install kan_gpt
```

## Usage

```py
from kan_gpt import BaseClass
from kan_gpt import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m kan_gpt
#or
$ kan_gpt
```
